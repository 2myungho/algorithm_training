## 설계

하루를 하나의 turn으로 생각하고 시뮬레이션을 돌린다.

매일 작업을 수행했을 때 100이 넘은 경우 check 배열을 true로 변경한다.

첫번째 단계가 끝나야 다음 단계로 넘어갈 수 있으므로, 현재 단계가 이전에 배포되어 skip해도 되는지를 판별하고, 이번 turn에 끝난 경우 count를 증가시키며 skip해도 됨을 표시한다.
