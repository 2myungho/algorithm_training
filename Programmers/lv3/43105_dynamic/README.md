## 설계

### 삼각형의 표현

삼각형을 2차원 배열로, 다음과 같이 표현한다.

```sh
7
3 8
8 1 0
2 7 4 4
4 5 2 6 5
```

그리고 같은 크기의 dp배열을 생성한다.

```cpp
int dp[502][502]; // 최대 크기 500까지이므로

// 일반식은 다음과 같다.
dp[i][j] = max(dp[i - 1][j] + triangle[i][j], dp[i - 1][j - 1] + triangle[i][j]);
```
