## 정리

내 코드 실행시간 18ms
제일 짧은 시간 5ms
입 출력 속도를 줄인뒤 시간 6ms

```cpp
ios_base ::sync_with_stdio(false);
cin.tie(NULL);
cout.tie(NULL);
```

입력받은 값의 배수를 구해가며 측정 범위 내에 들어오는지 확인하면 된다.
% 연산을 이용한 방법으로 풀수 없는지 의문.
cin, cout 의 속도를 개선했더니 12ms 정도나 줄일 수 있었다.
앞으로도 자주 써야겠다.
