## 설계

### 두 번째로 큰 큰 약수

N - (N의 두 번째로 큰 약수) 를 구하는 문제이다.

여기서 N이 2인 경우 최소 1을 빼야함 에유의한다.

두번째로 큰 약수는 N보다 작은 수이고, N / (두 번째로 작은 약수) 이다.

따라서 i=2부터 sqrt(N)+1 까지 증가하며 두 번째로 큰 약수를 갱신해나간다.
