# 2884. 알람 시계

## 설계

### 시간 복잡도

O(1) 이다.

### 공간 복잡도

시간은 24 이하, 분은 60 이하의 정수이므로 int형으로 충분하다.

### 시간의 계산

시간의 계산을 쉽게 하기 위해 모든 단위를, 최소단위로 통일해 변환 후 계산한다.

이 문제에서는 분 단위로 입력받은 시와 분을 변환 후 계산한다.

이 때 45분을 빼고나서 음수가 되는 경우에는 24시간 만큼을 더해준다.

그리고 각 단위별로 시와 분을 다시 계산한다.

```cpp
total = H * 60 + M - 45;
if (total < 0) total += 24 * 60;

H = total / 60;
M = total % 60;
```
