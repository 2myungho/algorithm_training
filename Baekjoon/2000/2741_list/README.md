## 설계

### 시간 복잡도

N은 최대 100,000 까지이다.

N까지 출력할 경우 시간 복잡도는 O(N) 이다. 이는 제한시간 내에 충분하다.

### 공간 복잡도

N은 최대 100,000 까지 이므로 int형으로 선언한다.
