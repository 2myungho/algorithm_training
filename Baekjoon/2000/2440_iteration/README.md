## 설계

### 시간복잡도

N은 최대 100 이고, 이중 반복문을 이용하므로 최대 10,000 번 연산한다.

이는 제한시간 1초 내에 충분하다.

### 이중 반복문

N, N-1, N-2, ... , 1번 별을 출력해야 하므로 다음과 같은 방법을 사용한다.

```cpp
for (int i = N; i > 0; i--)
  for (int j = 0; j < i; j++)
```
