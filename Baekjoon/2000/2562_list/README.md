## 설계

### 시간 복잡도

숫자의 갯수는 9개이다. 최대 값을 찾는 경우 N번만에 수행할 수 있으므로 제한시간 내에 충분하다.

### 공간 복잡도

100 이하의 자연수만 주어지므로 int형으로 선언한다.

### 순회

배열을 입력받고, 첫번째 숫자를 최대값으로 설정한 뒤 update 한다.

```cpp
for (int i = 0; i < 9; i++) {
  if (array[i] > maxValue) {
    maxValue = array[i];
    maxIndex = i;
  }
}
```
