## 설계

### N이 5의 배수인지 아닌지 여부에 따라 판별

N이 5의 배수인 경우 5킬로그램으로 분배가 가능하므로 남은 N은 0이된다.

위 조건을 만족하지 못한 경우, N이 3보다 클 때 N-=3을 수행한다.
(3킬로그램으로 분배)

## 정리

|      0       |       0        |

## 고생한 점

### 15 단위로 나누어 생각

3,5,6,8,9,10,11 ... 등등 15 이하에서 가능한 조합의 경우 숫자째로 빼서 답을 구하도록 구현

이 경우 16이 나오는 경우 오답이 출력됨
