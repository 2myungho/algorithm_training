## 설계

### 분기 처리

각 크로아티아 글자의 첫글자가 나올 때마다 분기처리를 한다

- c=
- c-
- dz=
- d-
- lj
- nj
- s=
- z=

d가 나온 경우, d-를 먼저 검사하고 dz=를 검사한다.

이때 index를 크로아티아 글자수만큼 증가시켜줘야 하므로,
문자 검사 전 index의 유효 범위를 검사한다.
