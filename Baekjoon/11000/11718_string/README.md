## 설계

### 시간복잡도

입력은 최대 100줄 이므로 최악의 경우 시간복잡도는 O(100) 이다.

중간에 공백이 있기 때문에 입력에 유의해야한다.

### getline의 이용

getline(cin, string)을 이용하면 한 줄을 통째로 읽을 수 있다.

이를 계속 반복하기 위해 while을 이용한다.

```cpp
while (getline(cin, line)) {
  // do something
}
```
