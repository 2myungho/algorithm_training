## 설계

### 이항계수(파스칼의 삼각형)

파스칼의 삼각형을 이용한다.

nCk = (n-1)C(k-1) + (n-1)Ck

```cpp
int combi(int n, int k) {
  if (k == 0 || n == k) return 1;

  // case_choose + case_not_choose
  return combi(n - 1, k - 1) + combi(n - 1, k);
}
```

### 메모이제이션

n,k쌍이 중복되는 경우가 많으므로, 시간복잡도를 줄이기 위해선 메모이제이션이 필요하다.
