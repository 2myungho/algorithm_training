## 설계

### 시간복잡도

테스트케이스 길이의 제한이 존재하지 않는다.

일반적으로 제한시간 내에 통과가 가능함이 보장된다.

### 공간복잡도

각 변의 길이는 최대 30,000 이하 이므로 int형으로 선언한다.

세 변의 길이가 오름차순으로 정렬 후 조건에 따라 분기한다.

```cpp
if (arr[2] * arr[2] == arr[1] * arr[1] + arr[0] * arr[0]) {
  cout << "right\n";
} else {
  cout << "wrong\n";
}
```
