## 설계

### 약수와 배수

두 수 A와 B를 입력받았을 때,

1. A 혹은 B가 0인경우
2. A % B == 0 인 경우
3. B % A == 0 인 경우
4. 이도저도 아닌경우

이렇게 4가지 경우로 분류할 수 있다.
