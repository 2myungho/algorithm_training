## 설계

### 알파벳 배열 alpha

각 알파벳의 index를 담을 배열 alpha를 선언한뒤 -1로 초기화한다.

```cpp
int alpha['z' - 'a' + 1];

memset(alpha, -1, sizeof(alpha));
```

입력받은 단어를 순회하며 알파벳이 나타난 경우 테이블을 갱신한다.

이때 해당 알파벳의 index가 -1 인 경우에만 갱신한다.
