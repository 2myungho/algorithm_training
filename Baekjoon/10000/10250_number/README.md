## 설계

### 순회

호텔의 방을 배열로 나타낸다면 다음과 같다.

```sh
rooms[H][W]
```

호텔 방의 우선 순위는 다음과 같이 순회한 결과이다.

```cpp
for (int h = 1; h <= H; h++) {
  for (int w = 1; w <= W; w++) {
    // do something
  }
}
```

N번째 까지 순회하며, N번째일때 호수를 출력한다.
