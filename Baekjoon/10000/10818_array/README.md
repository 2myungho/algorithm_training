## 설계

### 시간 복잡도

N은 최대 1,000,000 까지이다.

모든 수 중에서 최소, 최대를 구하는 데 걸리는 시간 복잡도는 N 이므로 제한시간 내에 충분하다.

### 공간 복잡도

입력받는 모든 수는 절대값이 1,000,000 이하이다. 따라서 int형으로 충분하다.

### 입력과 동시에 비교

```cpp
for (int i = 0; i < N; i++) {
  cin >> temp;
  minValue = min(minValue, temp);
  maxValue = max(maxValue, temp);
}
```
