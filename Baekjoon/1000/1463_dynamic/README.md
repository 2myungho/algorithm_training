## 설계

### 일반식

```cpp
dp[n] = min(dp[n-1], dp[n/2], dp[n/3]) + 1
// n이 2 혹은 3의 배수가 아닌경우
dp[n] = dp[n-1] + 1
```

위 일반식을 이용한다.

초기값은 다음과 같다

```cpp
dp[1] = 0;
dp[2] = 1;
dp[3] = 1;
```
