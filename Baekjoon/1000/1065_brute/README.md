# 1065. 한수

[링크](https://www.acmicpc.net/problem/1065)

정답률 : \_48%

## 설계

### 숫자를 문자열로 변환, 이전 자리수와 비교

```cpp
int diff = number[1] - number[0];
```

위와 같이 diff를 먼저 구하고, 각 연속된 자리의 차이를 diff와 비교한다.

### 1~9까지는 1자리인 문제

1~9까지는 모두 한수이므로

```cpp
answer = min(9, N);
```

으로 answer를 초기화 한 뒤 시작했다.

그 이후 for문을 10 부터 시작.

## 정리

내 코드 : 0 ms;

## 고생한 점

한 자리 숫자를 처리하는 부분을 예외처리할 때 if문을 안쓰고 구현한 점.
