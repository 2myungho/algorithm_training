## 설계

### 에라토스테네스의 채

에라토스테네스의 채를 이용해 풀이

1,000,000 까지의 소수를 모두 구하고 풀었을 때 12ms 소요.

입력받은 수 E (마지막 수) 까지 소수를 모두 구하고 풀었을 때도 12ms 소요.
