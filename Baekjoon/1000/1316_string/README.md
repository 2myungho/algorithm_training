## 설계

### 이전 문자를 기억

연속된 문자의 경우 무시해야 하기 때문에, 이전 문자와 같은지 다른지를 판단해야한다.

이전 문자와 같은경우 그냥 넘어가므로, 맨 처음 문자를 before로 설정하고
0번째 부터 순회한다.

```cpp
int before = word[0];
alpha[word[0] - 'a'] = true;
```

현재 문자와 이전 문자가 다른경우에 분기처리를 함

- 이전에 나온 문자인 경우 false
- 이전에 나온 문자가 아닌경우 before을 갱신하고 순회
