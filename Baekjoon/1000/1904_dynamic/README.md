## 설계

### 일반식

각 경우의 수를 순서대로 나열하면 다음과 같다

```sh
1 : 1
2 : 2
3 : 3
4 : 5
5 : 8
...
```

따라서 일반식은 다음과 같이 구할 수 있다.

```cpp
dp[i] = dp[i - 1] + dp[i - 2];
```
