## 설계

### 수식

```sh
A + Bx < Cx

A < (C-B)x
```

가 되는 x를 구한다.

따라서

```sh
A/(C-B) < x
```

인 x를 구하면 된다.

### 예외상황

- C-B가 0인 경우
- x가 음수인 경우
