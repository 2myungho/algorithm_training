### 자료구조

소숫점을 출력하기 위해 a와 b는 정수형이 아닌 실수형으로 입력받는다.

또한 출력시 소숫점 10번째 자리까지 출력하기 위해 다음 방법을 이용한다.

printf 이용시

```cpp
printf("%.10f", number);
```

cout 이용시

```cpp
cout.precision(10);
cout << number;
```
