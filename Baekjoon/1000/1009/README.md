## 설계

### 시간 복잡도

브루트 포스 방법으로 하는 경우 시간 복잡도는

```cpp
5 * 1,000,000
```

따라서 제한시간 1초 내에 충분하다.

### 공간 복잡도

100 이하의 자연수를 1,000,000 번 곱하는 경우 수가 매우 크므로, 1의 자리만 계산함

### 1의 자리만 계산

매번 연산을 수행할 때마다 1의 자리만 연산한다
