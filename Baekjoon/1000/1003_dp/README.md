## 설계

### 메모이제이션

fibonacci(n) = fibonacci(n‐1) + fibonacci(n‐2) 을 이용한다.

일반적인 피보나치 함수에서, 수열(숫자)를 리턴하는 것이 아닌
0과 1의 호출 횟수를 리턴하기 위해 구조체를 설정한다.

```cpp
struct counts {
  long long zero, one;
};
```

메모이제이션을 위한 dp를 선언

```cpp
counts fibo[41];
```

초기값

```cpp
fibo[0] = {1, 0};
fibo[1] = {0, 1};
```

이후 일반식을 이용해 구한다.
